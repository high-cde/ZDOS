# ZDOS: The Unified Ecosystem

Welcome to the **ZDOS Unified Repository**. This monorepo consolidates the entire suite of Z-GENESIS and ZDOS technologies into a single, cohesive framework.

## 🚀 Overview

ZDOS is a next-generation decentralized ecosystem combining high-performance neural computing, autonomous agent management, and a quantum-ready operating system.

## 📂 Project Structure

- **`core/`**: The intelligence layer.
  - `cortex/`: Neural routing and signal processing.
  - `aaak/`: Autonomous Agent Access Kernel.
  - `memory/`: High-scoring AI memory systems.
- **`os/`**: The foundation.
  - `kernel/`: Core OS kernel.
  - `ghostnet/`: Quantum Ghostnet OS components.
- **`network/`**: The infrastructure.
  - `nodes/`: Distributed node management.
  - `dsn/`: Distributed Service Network logic.
- **`interface/`**: The interaction layer.
  - `cli/`: Unified command-line interface (`zgenctl`).
  - `web/`: Live dashboard and visualization.
  - `cloud/`: Enterprise cloud portal.
- **`dev/`**: The toolkit.
  - `zen/`: Programming hub and SDKs.

## 🛠 Getting Started

To initialize the entire ecosystem, run the master build script:

```bash
./scripts/setup_all.sh
```

For individual module documentation, please refer to the `docs/` directory or the README within each submodule.
